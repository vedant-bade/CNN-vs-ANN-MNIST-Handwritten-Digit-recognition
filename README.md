# Handwritten Digit Classification using CNN Vs. ANN

## Overview

The main objective of the Project is to show the difference between the accuracy and predictions between a CNN model and a ANN model.
This project uses a Convolutional Neural Network (CNN) built with TensorFlow/Keras to classify handwritten digits from the MNIST dataset.

## Features

- CNN Architecture
- ANN Architecture
- Image Normalization
- Model Training
- Accuracy Evaluation
- Predictions on Test Images

## Dataset

MNIST Dataset

- 60,000 Training Images
- 10,000 Test Images

## Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Results

CNN Test Accuracy: 99.25%
MultiLayer ANN Test Accuracy: 97.53%
Single Layer Perceptron Test Accuracy: 90.88%
